# Chamomile AR Mobile Application

An augmented reality (AR) mobile application that visualizes a 3D chamomile flower using image recognition.
The application provides interactive educational content, including quizzes and multimedia points of interest.

---

## Tech Stack

* Unity (C#)
* Blender (3D modeling)
* Image Recognition (AR)
* Android

---

## Features

* Image recognition to trigger AR content
* 3D chamomile model created in Blender and integrated into Unity
* Interactive quiz functionality
* Informational content about chamomile
* 3 Points of Interest (POIs) including audio, images, and video

## How It Works

1. Open the application
2. Scan the image target shown above
3. The 3D chamomile model will appear in AR
4. Interact with educational content and multimedia elements
