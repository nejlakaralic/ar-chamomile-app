# Chamomile AR Mobile Application

An augmented reality (AR) mobile application that visualizes a 3D chamomile flower using image recognition.
The application provides interactive educational content, including quiz and multimedia points of interest.

---

## Tech Stack

* Unity (C#)
* Blender (3D modeling)
* Image Recognition / AR
* Android

---

## Features

* Image recognition to trigger AR content
* 3D chamomile model created in Blender and integrated into Unity
* Interactive quiz functionality
* Informational content about chamomile
* 3 Points of Interest (POIs) including:

  * Audio
  * Images
  * Video

---

## How It Works

1. The user scans a predefined image
2. The application recognizes the image
3. A 3D chamomile model is displayed in real-world space
4. The user can interact with educational and multimedia content
